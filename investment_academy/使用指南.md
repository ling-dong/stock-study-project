## 一、首次安装

```bash

cd D:\stock_market\investment_academy

# 2. 安装 Python 依赖
pip install -r requirements.txt

# 3. 安装前端依赖
```

## 二、启动系统（前后端分离模式）

需要**同时打开两个终端窗口**：

### 终端 1 — 启动后端 API

```bash
cd D:\stock_market\investment_academy
uvicorn backend.main:app --reload --port 8000
```

启动成功后看到：

```
INFO:     Uvicorn running on http://127.0.0.1:8000 (Press CTRL+C to quit)
```

> 后端 API 文档自动生成在 http://localhost:8000/docs

### 终端 2 — 启动前端

```bash
cd D:\stock_market\investment_academy\frontend
npm run serve
```

启动成功后看到：

```
  App running at:
  - Local:   http://localhost:8080/
```

### 打开浏览器访问

> **http://localhost:8080**

## 四、交易沙盒使用流程

```
① 选择 ETF → ② 选时间框架 → ③ 选起始日期（可选）
       ↓
④ 点击「开始交易」
       ↓
⑤ 观察历史 K 线（30根背景 + 黄色"交易起点"标记）
       ↓
⑥ 每次决策：
   - 点击「下一天」推进行情
   - 输入股数 → 点击「买入」（A股 100股/手）
   - 点击「全卖」或输入数量「部分卖」
       ↓
⑦ 数据走完后自动显示绩效报告
```

### 画线工具

1. 点击控制栏的 **📏 画线** 按钮进入画线模式
2. 在 K 线图上点击两根蜡烛 → 自动连线（趋势线/支撑压力线）
3. 可画多条线，点击 **清空** 删除所有画线

## 七、常见问题

### Q: 前端页面打开但数据不显示

**检查后端是否在运行**：浏览器访问 http://localhost:8000/api/health 应返回 `{"status":"ok"}`。如果没有，说明后端未启动。

### Q: 后端启动报错 `[WinError 10013]` / `address already in use`

端口 8000 被占用。先停掉旧进程：

```bash
taskkill /F /IM python.exe
```

### Q: K 线图不显示

刷新页面。如果仍不显示，检查浏览器控制台是否有红色报错。

### Q: 选起始日期后图表和历史数据对不上

已修复 — 图表会从历史 30 根 K 线处开始显示，黄色虚线标记沙盒交易起点。

### Q: ETF 数据浏览器没有数据表

图表下方已增加「原始数据（最近 30 条）」表格。

## 九、项目结构

```
investment_academy/
├── core/                 共享核心（Be→Fa→St 共用）
│   ├── models/           数据模型（dataclass）
│   ├── db/               SQLite 持久化（6 表 + CRUD）
│   ├── bridge/           SPAS 数据读取适配器
│   └── engine/           核心引擎（沙盒 + 内容加载）
├── backend/              FastAPI 后端（36 个 API 端点）
│   ├── main.py           入口 + CORS
│   ├── schemas.py        Pydantic 模型
│   └── routers/          8 个路由模块
├── frontend/             Vue2 前端（8 个页面）
│   └── src/
│       ├── views/        页面组件
│       ├── components/   KLineChart / QuizWidget / MarkdownViewer / ProgressBar
│       └── api/          axios 封装
├── streamlit_app/        Streamlit 应用（备选）
│   ├── app.py            入口
│   ├── pages/            页面
│   └── widgets/          Streamlit 专用组件
├── content/              学习内容（34 章 + 170 道测验题）
├── tests/                56 个自动化测试
└── requirements.txt      Python 依赖
```
