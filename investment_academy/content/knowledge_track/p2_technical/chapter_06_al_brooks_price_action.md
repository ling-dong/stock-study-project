# 第六章：Al Brooks 价格行为理论

## 6.1 Al Brooks 是谁？

Al Brooks 是一位职业交易员和畅销书作家，他的《价格行为交易》三部曲（趋势、交易区间、反转）被公认为价格行为分析的经典。与 Wyckoff 关注大资金运作逻辑不同，Brooks 的体系聚焦于**逐根 K 线的微观博弈**，适合日内和短线交易者。

> 💡 Brooks 的核心哲学：**价格行为是唯一真实的信息源。** 所有指标都是价格的衍生品，市场参与者的一切决策最终都反映在价格上。

## 6.2 三种市场结构

Brooks 将市场定义为三种状态：

| 状态 | 特征 | 操作原则 |
|------|------|----------|
| **趋势（Trend）** | 价格持续朝一个方向移动，回调浅且短暂 | 只顺势做，不逆势 |
| **交易区间（Trading Range）** | 价格在支撑和阻力之间波动，方向不明确 | 高抛低吸，或等待突破 |
| **反转（Reversal）** | 趋势转为交易区间或反方向趋势 | 识别反转信号，及时离场 |

> 🔑 **80% 的价格行为发生在交易区间中。** 强趋势只占约 20% 的时间。大多数交易者的亏损来自在交易区间中用趋势策略操作。

## 6.3 K 线的角色：信号棒与入场棒

Brooks 对每根 K 线赋予功能角色：

### 信号棒（Signal Bar）

信号棒是触发交易决策的 K 线。它告诉你"方向可能正在变化"。

```
买入信号棒的特征：
  ✅ 收盘在最高位附近（close_position > 0.6）
  ✅ 低点比前几根 K 线低 → 卖方尝试压价但失败
  ✅ 收盘时买方完全拿回控制权
  ✅ 实体不需要很大，但收盘位置要好

卖出信号棒的特征：
  ✅ 收盘在最低位附近（close_position < 0.4）
  ✅ 高点比前几根 K 线高 → 买方尝试推高但失败
  ✅ 收盘时卖方拿回控制权
```

### 入场棒（Entry Bar）

入场棒是信号棒之后的那根 K 线——你在这根 K 线上实际执行交易。

```
入场规则：
  止损买单（Stop Entry）：
    在信号棒高点之上 1 tick 挂买单
    → 如果下一根 K 线的价格突破信号棒高点 → 自动成交
    → 表示买方力量已证实

  限价单（Limit Entry）：
    在信号棒低点附近挂限价买单
    → 价格回调到支撑位时自动买入
    → 成本更好，但可能不成交
```

> ⚠️ A股 T+1 制度下，"下一根 K 线入场"的逻辑需调整为开盘入场。日线级别，信号棒=昨天，入场棒=今天开盘。

## 6.4 二次入场（Second Entry）— 最可靠信号

这是 Brooks 体系中最核心的概念之一：

```
第一次回调试探 → 买方推回去（不确认）
第二次回调试探 → 再次被推回去
    ↓
这是一个二次入场信号 → 趋势可能已经恢复
```

```
趋势中的二次回调：
  
  上升趋势
    ↗
     ╲ 回调1（卖方试了一次，失败）
      ↗ 反弹
       ╲ 回调2（卖方再试，再次失败）← 这里！二次入场
        ↗ 趋势恢复，强势上涨
```

**为什么二次入场比一次入场更好？**
- 第一次回调可能是趋势反转的开始
- 第二次回调确认卖方没有力量
- 两次失败的逆势尝试 = 顺势力量更强

## 6.5 趋势棒 vs 十字星

Brooks 对每根 K 线做了一个关键的二元分类：

| 类型 | 定义 | 市场含义 |
|------|------|----------|
| **趋势棒** | 开盘到收盘有明确的实体，实体 > 影线 | 一方掌控，方向明确 |
| **十字星** | 开盘≈收盘，实体极小，影线较长 | 多空拉锯，方向犹豫 |

> **连续的趋势棒 = 强趋势。趋势棒后出现十字星 = 趋势减弱。** 这是 Brooks 判断趋势强度的核心方法，无需任何指标。

## 6.6 通道理论

Brooks 观察到价格运动倾向于形成通道：

```
上升通道：
  ↗→↘→↗→↘→↗   （两条平行趋势线）
  上轨 = 阻力，下轨 = 支撑

通道规则：
  - 触碰上轨 → 短线减仓（但不做空！顺势永远不做空）
  - 触碰下轨 → 加仓机会
  - 通道变陡 → 趋势加速（可能进入高潮阶段）
  - 通道变平 → 趋势减弱（可能转为交易区间）
```

## 6.7 测量目标（Measured Move）

```
等距测量法则：
  上一波上涨幅度 = L1
  回调幅度 = L2
  如果趋势继续，下一波目标 ≈ L1

  即：如果第一波涨了 5 元，回调 2 元后恢复，
  下一波的目标是 5 元（从回调低点算起）
```

> 注意：这里的 "测量目标" 不是 "缺口（Gap）"。缺口是指 K 线之间的价格跳空，而 Measured Move 是纯粹用价格波幅估算目标位。

## 6.8 量价关系（Brooks 风格）

Brooks 不使用成交量指标，但关注**价格走势本身的量价关系**：

| 模式 | 解读 |
|------|------|
| 大趋势棒 + 小回调 | 趋势强，顺势方完全掌控 |
| 小趋势棒 + 大回调 | 趋势弱，可能转为震荡 |
| 连续十字星 | 市场完全没方向，不做交易 |
| 窄幅横盘后突然放量突破 | 最可靠的入场信号之一 |

> 🔑 "如果你看不懂当前的价格行为，那就空仓。市场总是在那里，不会跑掉。"

## 6.9 Brooks 与 Wyckoff 的对比

| 维度 | Wyckoff | Al Brooks |
|------|---------|-----------|
| 分析单位 | 大结构（积累/派发区，数月） | 小结构（逐根 K 线，数日到数周） |
| 核心关注 | 聪明钱的供需博弈 | K 线本身的博弈结果 |
| 信号类型 | H2/L2/FB（形态） | 信号棒 + 二次入场 |
| 适用周期 | 日线/周线为主 | 日线/60分/5分为主 |
| 胜率 | H2~55%, FB~45% | 二次入场 ~60%+ |
| 学习曲线 | 陡峭（主观性强） | 中等（规则更机械） |

> 💡 **两者互补而非互斥**。Wyckoff 看大局（"这里是不是积累区？"），Brooks 看细节（"这根 K 线是趋势棒还是十字星？"）。

## 6.10 实战：Brooks 入场 Checklist

```
入场前逐项确认：
□ 当前是趋势还是交易区间？
  → 趋势：只顺势做
  → 交易区间：高抛低吸或等待

□ 信号棒收盘位置好吗？
  → 买入信号：close_position > 0.6
  → 卖出信号：close_position < 0.4

□ 这是二次入场吗？
  → 如果是：胜率更高，可以正常仓位
  → 如果不是：减半仓位

□ 信号棒是趋势棒还是十字星？
  → 趋势棒信号 > 十字星信号
  → 十字星信号 = 等下一根确认再入场

□ 通道位置合理吗？
  → 在通道下轨附近买入（成本好）
  → 不在通道上轨追买（风险高）
```
