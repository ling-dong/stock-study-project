# 第五章：多时间框架分析

## 5.1 为什么需要多时间框架？

单一时间框架的 K 线图存在致命缺陷：**任何时间框架上的信号，在更大时间框架的趋势面前都可能失效。**

> 5分钟线出现完美的 H2 买入信号，但日线正在熊市中 → 这个 H2 大概率会失败。

SPAS 的 Fusion 层使用了 **4 个时间框架**（月/周/日/60分）的投票融合机制，各框架的权重根据波动率动态调整。

## 5.2 三层分析法

| 层次 | 时间框架 | 角色 | 不允许做的事 |
|------|----------|------|-------------|
| **战略层** | 日线 | 决定方向（多/空/观望） | — |
| **战术层** | 60分钟线 | 寻找入场时机 | 逆战略层方向 |
| **执行层** | 15/5分钟线 | 确定精确买卖点 | 无战术层信号时单独行动 |

### 铁律

> **执行层永远服从战术层，战术层永远服从战略层。日线看空时不做多，哪怕 5 分钟线出现了 10 个买入信号。**

## 5.3 框架间的三种关系

### 共振（最强信号）

所有框架方向一致。例如日线多头排列 + 60分回调到位 + 15分放量突破 → **高确定性入场**。

### 背离（预警信号）

不同框架方向相反。例如日线还在涨但 60 分已经出现顶部结构 → **减仓预警**。背离是所有重大转折的前兆。

### 无关系（噪音）

框架之间没有明确的方向关系，说明市场处于混沌状态 → **不做交易，观望**。

## 5.4 SPAS 多框架融合

SPAS 的 Fusion 层给每个时间框架分配动态权重：

```
总信号 = Σ (框架i的预测 × 框架i的权重)
权重 = f(框架i的近期准确率, 框架i的ADX, 波动率)
```

当前框架的 ADX 越高，其投票权重越大——强趋势框架的意见更重要。

## 5.5 实战 Checklist

```
买入前检查：
□ 日线：EMA(20) 斜率方向？
□ 日线：ADX > 18？
□ 60分：出现了 H2/FB/突破信号？
□ 15分：放量确认？收盘位置 > 0.6？

如果以上有任何一个不满足 → 不交易
如果全部满足 → 按计划仓位入场
```

## 本章要点

| 概念 | 说明 |
|------|------|
| 战略/战术/执行三层 | 日线定方向、60分找时机、15分定点位 |

| 背离 | 框架方向不一致 = 预警 |
| 铁律 | 小框架永远服从大框架 |
