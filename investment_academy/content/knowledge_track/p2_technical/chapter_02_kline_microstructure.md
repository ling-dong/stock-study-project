# 第二章：K线微观结构

## 3.1 什么是 K 线微观结构？

在 P1 中我们学习了 K 线的基本构造。现在深入到一个层次：**微观结构分析**——通过精确的数学公式解读单根 K 线传递的多空博弈信息。

每一根 K 线都是多空双方在这一时间段内博弈的完整记录。SPAS 系统将这种博弈量化为 **6 个特征因子**。

## 3.2 六维 K 线特征因子

### 因子 1：实体比（body_ratio）

```
body_ratio = |Close - Open| / (High - Low)
范围: [0, 1]
```

| 值 | 含义 | 博弈解读 |
|----|------|----------|
| >0.7 | 大实体 | 一方压倒性优势，趋势明确 |
| 0.3-0.7 | 中等实体 | 博弈进行中，方向有偏向 |
| <0.3 | 小实体/十字星 | 多空拉锯，势均力敌 |
| ≈0 | 十字星（Doji） | 完全均衡，往往预示反转 |

> 🔑 实体比大 = 共识强 = 趋势不容易反转。实体比小 = 分歧大 = 反向可能性高。

### 因子 2：收盘位置（close_position）

```
close_position = (Close - Low) / (High - Low)
范围: [0, 1]
```

这是最重要的单因子之一。它告诉你**谁在收盘时掌控局面**：

| 值 | 含义 |
|----|------|
| >0.6 | 买方强势收盘，次日大概率高开 |
| 0.4-0.6 | 中性收盘 |
| <0.4 | 卖方强势收盘，次日大概率低开 |

> 💡 **收盘位置比实体比更重要**。一根小实体阳线如果收盘在最高点（close_position≈1），比一根大实体但收盘不在最高的阳线更有看涨意义。

### 因子 3：上影线比例（upper_shadow）

```
upper_shadow = (High - max(Open, Close)) / (High - Low)
范围: [0, 1]
```

| 值 | 含义 |
|----|------|
| >0.4 | 长上影线 → 上方抛压重，价格冲高后被压回 |
| 0.1-0.4 | 正常影线 |
| <0.1 | 光头 → 买方掌控全局（如果是上涨） |

> ⚠️ 上涨趋势中出现长上影线 = 高位出货信号。下跌趋势中出现长上影线可能是诱多。

### 因子 4：下影线比例（lower_shadow）

```
lower_shadow = (min(Open, Close) - Low) / (High - Low)
范围: [0, 1]
```

| 值 | 含义 |
|----|------|
| >0.4 | 长下影线 → 下方支撑强，价格下探后被拉起 |
| 0.1-0.4 | 正常影线 |
| <0.1 | 光脚 → 卖方掌控全局（如果是下跌） |

> 💡 下跌趋势中出现长下影线 = 抄底资金介入信号。上涨趋势中长下影线可能是洗盘。

### 因子 5：趋势棒方向（trend_bar）

```
trend_bar = +1 (Close > Open, 阳线)
          = -1 (Close < Open, 阴线)
          =  0 (Close ≈ Open, 十字星)
```

虽然简单，但连续的趋势棒方向可以判断多空力量对比。SPAS 用最近 N 根 K 线中阳线的比例来判断市场偏向。

### 因子 6：涨跌停状态（limit_status，A股特有）

```
limit_status = ±1 (触及涨跌停板)
             =  0 (正常交易)
```

A 股有 ±10%（主板）或 ±20%（科创/创业）涨跌幅限制。触及涨跌停是极端信号：

| 情况 | 含义 |
|------|------|
| 涨停板 | 买方极度强势，惜售明显，次日易高开 |
| 跌停板 | 卖方恐慌性抛售，次日大概率继续低开 |
| 连续涨停/跌停 | 极端情绪，趋势自我强化 |

> ⚠️ **边界情况**：当 High = Low（如停牌、无成交）时，4个比率的除数均为 0，公式无定义。此时应跳过或标记为无效数据。

## 3.3 实战综合运用

单因子信号不可靠，需要综合判断。以下是一个典型的**看涨组合**：

```
✅ body_ratio > 0.5       大实体
✅ close_position > 0.7   收盘在高位
✅ upper_shadow < 0.1     几乎没有上影线（光头）
✅ lower_shadow < 0.1     几乎没有下影线
→ 标准的大阳线，买方完全掌控，次日大概率继续涨
```

看跌组合：

```
✅ body_ratio > 0.5       大实体
✅ close_position < 0.3   收盘在低位
✅ upper_shadow < 0.1     光头
✅ lower_shadow < 0.1     光脚
→ 标准的大阴线，卖方完全掌控，次日大概率继续跌
```

反转警告组合：

```
✅ body_ratio < 0.2       小实体（十字星类）
✅ close_position ≈ 0.5   收盘在中间
✅ upper_shadow > 0.3     长上影线
✅ lower_shadow > 0.3     长下影线
→ 经典十字星，多空势均力敌，次日方向决定短期走势
```

## 3.4 微观结构的局限

1. **单根 K 线没有上下文就是噪音**。必须结合趋势、位置、成交量综合判断
2. **异常影线可能来自流动性问题**而非博弈意图（尾盘/集合竞价容易出现）
3. **不同时间框架的微观结构含义不同**。日线的十字星和 5 分钟线的十字星完全不同

## 本章要点

| 概念 | 说明 |
|------|------|
| 实体比 | 博弈强度的量化，越大越确定 |
| 收盘位置 | 最重要的单因子，收盘时谁在掌控 |
| 上影线 | 盘中冲高被压回 = 抛压 |
| 下影线 | 盘中下探被拉起 = 支撑 |
| 综合判断 | 6 因子组合 > 单因子信号 |
