# 第三章：均线与ADX

## 4.1 均线（Moving Average）的本质

均线是过去 N 个周期收盘价的算术平均值。它本质上是**趋势的平滑化**——滤除短期噪音，暴露趋势方向。

```
SMA(N) = (C₁ + C₂ + ... + Cₙ) / N

EMA(N) 平滑系数 α = 2/(N+1)
  EMA(t) = P(t) × α + EMA(t-1) × (1-α)
  （递归定义，近期价格权重更高）
```

SPAS 系统偏好 **EMA**，因为它在保持平滑的同时对近期变化更敏感。

## 4.2 常用均线周期

| 周期 | 含义 | 用途 |
|------|------|------|
| EMA(5) | 一周线 | 超短线趋势，对价格变化最敏感 |
| EMA(10) | 半月线 | 短线趋势 |
| EMA(20) | 月线 | **核心趋势线**，SPAS 市场状态机的基准 |
| EMA(60) | 季线 | 中期趋势支撑/压力 |

> 🔑 EMA(20) 是职业交易者的默认基准。价格在 EMA20 之上 = 短期偏多，之下 = 短期偏空。

## 4.3 均线形态

### 多头排列（牛市信号）
```
EMA(5)  >  EMA(10)  >  EMA(20)  >  EMA(60)
  ↑           ↑           ↑           ↑
 短线         中线         趋势        长期
 贴近价格最近                                  贴近价格最远
```

价格在最上方，均线从上到下按短期到长期排列。这是最强的看涨信号。

### 空头排列（熊市信号）
```
EMA(60)  >  EMA(20)  >  EMA(10)  >  EMA(5)
 长期         趋势         中线         短线
 贴近价格最远                                  贴近价格最近
```

价格在最下方，与多头排列完全相反。

### 均线缠绕（震荡信号）
均线交叉缠绕，没有明确的上下关系 → 市场无方向，不适合趋势交易。

## 4.4 金叉与死叉

| 信号 | 定义 | 可靠性 |
|------|------|--------|
| **金叉** | 短期 MA 上穿长期 MA | EMA(5) 上穿 EMA(20)：中高 |
| **死叉** | 短期 MA 下穿长期 MA | EMA(5) 下穿 EMA(20)：中高 |

> ⚠️ 均线交叉是**滞后信号**。金叉发生时价格往往已经涨了一段。均线交叉更适合用来确认趋势，而非捕捉精确的转折点。

## 4.5 ADX 深入

ADX（Average Directional Index）由 Welles Wilder 发明，综合了 +DI（上升方向线）和 -DI（下降方向线）：

### ADX 的组合信号

| ADX | +DI vs -DI | 含义 |
|-----|-----------|------|
| 上升 | +DI > -DI | 上升趋势加强 |
| 上升 | -DI > +DI | 下降趋势加强 |
| 下降 | 任意 | 趋势减弱，可能反转或进入震荡 |
| >40 | 任意 | 趋势过热，注意衰竭 |

### SPAS 系统使用 ADX 的方式

SPAS 的市场状态机用 ADX(14) 判断趋势强度：
- ADX < 18：状态维持 NEUTRAL，不产生交易信号
- ADX > 18 + EMA(20) 斜率向上 → BULL
- ADX > 18 + EMA(20) 斜率向下 → BEAR
- ADX > 25 + 趋势确认 → 加仓信号

## 4.6 实战：均线 + ADX 组合策略

```
入场条件：
  ✅ EMA(20) 斜率向上
  ✅ ADX > 18 且上升
  ✅ +DI > -DI
  
出场条件（任一触发）：
  ❌ ADX 从高位回落 > 5 点
  ❌ EMA(5) 下穿 EMA(10)
  ❌ 价格跌破 EMA(20) 且无法快速收回
```

## 本章要点

| 概念 | 说明 |
|------|------|
| 均线 | 趋势的平滑化，核心是 EMA(20) |
| 多头排列 | EMA5>EMA10>EMA20>EMA60，最强多头 |
| 金叉/死叉 | 确认信号，非精确定时 |
| ADX | 趋势强度计，越高趋势越强 |
| +DI/-DI | 趋势方向的辅助判断 |
