# 第四章：K线图入门

## 4.1 什么是 K 线图？

**K线图（Candlestick Chart）** 起源于日本江户时代的米市交易，由本间宗久发明，用于记录大米价格的波动。如今，它是全球金融市场最通用的价格图表形式。

每一根 K 线（蜡烛）记录了特定时间段内的四个核心价格：
- **开盘价（Open）** — 该时间段第一笔成交价
- **最高价（High）** — 该时间段内达到的最高价格
- **最低价（Low）** — 该时间段内达到的最低价格
- **收盘价（Close）** — 该时间段最后一笔成交价

## 4.2 K 线的解剖结构

```
        最高价 ──→ ┃  （上影线 / 上须）
                    ┃
        ┌──────┐   ┃
        │ 实体  │   ┃  ← 开盘价和收盘价之间的矩形
        └──────┘   ┃
                    ┃
        最低价 ──→ ┃  （下影线 / 下须）
```

### 阳线 vs 阴线

| 类型 | 特征 | 含义 | 常见颜色 |
|------|------|------|----------|
| **阳线** | 收盘 > 开盘 | 价格上涨，买方占优 | 红色/绿色 |
| **阴线** | 收盘 < 开盘 | 价格下跌，卖方占优 | 绿色/红色 |

> 🎨 中国习惯：红涨绿跌（A股）；国际习惯：绿涨红跌（美股）。在投资学院中，我们跟随 A 股习惯使用**红涨绿跌**。

## 4.3 关键形态解读

### 实体大小

| 实体特征 | 市场含义 |
|----------|----------|
| 长实体（大阳线/大阴线） | 单边力量极强，趋势明确 |
| 短实体 | 多空拉锯，方向不明确 |
| 十字星（开盘≈收盘） | 多空势均力敌，可能趋势反转 |

### 影线含义

| 影线特征 | 市场含义 |
|----------|----------|
| 长上影线 | 曾经冲高但被压回，**上方抛压大** |
| 长下影线 | 曾经下探但被拉起，**下方支撑强** |
| 光头光脚（无影线） | 极端强势，一面倒的行情 |

### 实战中常见的组合

```
锤子线（Hammer）：              吊颈线（Hanging Man）：
    ┃                                ┃
    ┃                                ┃
  ╺━┫  ← 小实体                   ╺━┫  ← 小实体
    ┃                                ┃
    ┃                                ┃
    ┃                                ┃
    ╹  ← 长下影线                    ╹  ← 长下影线
底部反转信号                      顶部反转信号
```

## 4.4 OHLCV 数据模型

每根 K 线背后的数据结构：

```python
{
    "trade_date": "2024-01-15",
    "open":  3.520,   # 开盘价
    "high":  3.580,   # 最高价
    "low":   3.480,   # 最低价
    "close": 3.550,   # 收盘价
    "volume": 15800000 # 成交量（股）
}
```

在投资学院中，我们使用 SPAS 系统的真实 A 股 ETF 数据，所有 K 线图都基于这个 OHLCV 格式。

## 4.5 时间框架

同一只股票可以有不同的 K 线周期：

| 时间框架 | 每根 K 线代表 | 适用场景 |
|----------|-------------|----------|
| 日线（Day） | 一天 | 中期趋势判断 |
| 60分钟线 | 一小时 | 日内波段 |
| 15分钟线 | 15分钟 | 短线进出 |
| 5分钟线 | 5分钟 | 极短线/日内交易 |
| 周线（Week） | 一周 | 长期趋势 |
| 月线（Month） | 一月 | 超长期大方向 |

> 🔑 **多时间框架分析**（MTF）是进阶技巧：日线看方向，60分钟找入场点，5分钟精确时机。

## 4.6 常见误区

| 误区 | 真相 |
|------|------|
| "阳线就一定代表强势" | 一根孤立的阳线意义不大，要看位置和上下文 |
| "十字星一定会反转" | 十字星只代表犹豫，不一定反转 |
| "影线越长越好/越坏" | 影线需要配合成交量和位置综合判断 |

## 本章要点

| 概念 | 说明 |
|------|------|
| K 线 | 记录 O/H/L/C 四个价格的可视化图表 |
| 阳线 | 收盘 > 开盘 → 买方主导 |
| 阴线 | 收盘 < 开盘 → 卖方主导 |
| 实体 | 开盘价和收盘价之间的矩形 |
| 影线 | 实体到最高/最低价的细线，反映极限价位 |
| OHLCV | Open/High/Low/Close/Volume 数据结构 |
| 时间框架 | 日线/60min/5min 等不同周期 |
