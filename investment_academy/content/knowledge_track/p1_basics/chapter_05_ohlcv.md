# 第五章：基本术语

## 5.1 价格相关

| 术语 | 英文 | 含义 |
|------|------|------|
| **开盘价** | Open | 每个交易日第一笔成交价格 |
| **收盘价** | Close | 每个交易日最后一笔成交价格（最重要的价格） |
| **最高价** | High | 当日达到的最高成交价 |
| **最低价** | Low | 当日达到的最低成交价 |
| **涨跌幅** | Change% | (当日收盘 − 昨日收盘) ÷ 昨日收盘 × 100% |
| **涨停** | Limit Up | 涨到当日允许的最高价 |
| **跌停** | Limit Down | 跌到当日允许的最低价 |

## 5.2 成交量与流动性

| 术语 | 含义 | 判断标准 |
|------|------|----------|
| **成交量** | 当日成交的股票数量（单位：股/手） | 越大越活跃 |
| **成交额** | 成交量 × 成交均价（单位：元） | 反映真金白银 |
| **换手率** | 成交量 ÷ 流通股本 × 100% | <1%冷清，3-5%正常，>10%异常活跃 |
| **量比** | 当日成交量 ÷ 前5日均量 | >1放量，<1缩量 |

> 🔑 "量在价先" — 成交量变化往往提前于价格变动。放量突破比缩量突破更可靠。

## 5.3 估值指标

| 指标 | 公式 | 含义 |
|------|------|------|
| **PE（市盈率）** | 股价 ÷ 每股收益 | 按当前盈利水平，多少年回本 |
| **PB（市净率）** | 股价 ÷ 每股净资产 | 市场价相对于账面价值的倍数 |
| **PS（市销率）** | 总市值 ÷ 营业收入 | 适用于尚未盈利的成长型公司 |
| **股息率** | 每股分红 ÷ 股价 × 100% | 持有股票的现金回报率 |

### PE 怎么理解？

```
PE = 20  →  如果盈利不变，需要 20 年通过利润收回投资成本
PE = 100 →  需要 100 年 → 市场预期未来盈利会大幅增长（也可能是泡沫）
PE = 5   →  只需要 5 年 → 市场认为该公司前景黯淡（也可能是低估机会）
```

> ⚠️ PE 不能跨行业比较！科技股 PE 天然高，银行股 PE 天然低，跨行业对比 PE 没有意义。

## 5.4 市值

| 分类 | 市值范围 | 特征 |
|------|----------|------|
| **大盘股** | >1000亿 | 流动性好，波动小，机构主导 |
| **中盘股** | 100亿~1000亿 | 平衡 |
| **小盘股** | <100亿 | 波动大，容易被资金炒作 |

> 💡 **市值 = 股价 × 总股本**。注意：股价高不代表公司大（茅台1700元但市值仅约2万亿），股价低不代表公司小。

## 5.5 行情术语

| 术语 | 含义 |
|------|------|
| **牛市** | 价格长期上涨趋势，市场乐观 |
| **熊市** | 价格长期下跌趋势，市场悲观 |
| **震荡市** | 价格在一定区间内来回波动，方向不明 |
| **反弹** | 下跌趋势中的短期上涨 |
| **回调** | 上涨趋势中的短期下跌 |
| **突破** | 价格涨破重要阻力位或跌破重要支撑位 |
| **回踩** | 突破后价格重新测试突破位置 |

## 5.6 订单类型

| 类型 | 说明 |
|------|------|
| **市价单** | 以当前最优价格立即成交，速度快但价格不可控 |
| **限价单** | 指定价格，只有达到目标价才成交，价格可控但可能不成交 |
| **止损单** | 触发价格后自动变成市价单卖出，用于限制亏损 |

> 新手建议：除非紧急情况，**始终使用限价单**。市价单在流动性差时可能以很差的价格成交。

## 5.7 仓位管理术语

| 术语 | 含义 |
|------|------|
| **仓位** | 目前持有股票占你总资金的比例 |
| **满仓** | 全部资金都买了股票 |
| **空仓** | 全部持有现金，没有股票 |
| **半仓** | 一半资金在股票，一半是现金 |
| **加仓** | 在已有持仓的基础上继续买入 |
| **减仓** | 卖出一部分持仓 |
| **止损** | 价格跌到预设位置，卖出以控制亏损 |
| **止盈** | 价格涨到预设目标，卖出锁定利润 |

## 本章要点

| 概念 | 说明 |
|------|------|
| OHLCV | 金融市场最基本的数据结构 |
| 换手率 | 衡量交易活跃度的核心指标 |
| PE（市盈率） | 估值的最常用指标，跨行业不能比 |
| 市值 = 股价 × 总股本 | 衡量公司体量 |
| 市价单 vs 限价单 | 新手建议用限价单 |
| 仓位管理 | 决定你承担多少风险的核心决策 |
