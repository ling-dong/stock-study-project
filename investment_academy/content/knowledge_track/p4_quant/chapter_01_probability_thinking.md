# 第一章：概率思维

## 1.1 投资的核心是概率，不是预测

> **没有人能准确预测明天的股价。** 如果有人声称能做到，要么在骗你，要么在骗自己。

职业交易员不谈"会不会涨"，而是谈"涨的概率有多大"和"涨多少的概率有多大"。这和天气预报员的思维一模一样：**不说"明天一定下雨"，而是"70% 的概率下雨"。**

## 1.2 概率思维的三个核心概念

### 概念一：基率（Base Rate）

在做任何预测之前，先问：**历史上，类似情况发生的概率是多少？**

```
例子：看到 H2 Setup 出现
→ 不去猜这次能不能成功
→ 先查历史：H2 在 100 次中出现，有 55 次成功
→ 基率 = 55%
```

SPAS 的 Rule Engine 对每个信号都附带了基率。这是一个天然的无偏基准——如果你觉得这次"肯定能成"，先看看基率，冷静一下。

### 概念二：贝叶斯更新

基率是起点，不是终点。每当你获得新的信息，就应该更新你的概率估计：

```
初始概率（基率）：55%（基于 H2 信号本身）

+ 成交量在回调中萎缩（质量因子）→ 更新到 60%
+ 日线 EMA(20) 斜率向上（趋势对齐）→ 更新到 65%
+ 当前板块也是热点（情绪共振）→ 更新到 70%
- 大盘指数在走弱（系统性风险）→ 回调到 63%

最终决策概率：63%
```

> 🔑 **贝叶斯思维 = 基于证据逐步调整信心**，而不是一个信号就全信或全不信。

### 概念三：收缩估计（Shrinkage Estimation）

当数据量不够大时，对极端值的估计要"收缩"回均值方向。

```
只有 5 次样本，胜率 80% → 收缩到 65%（因为样本太小，不可信）
有 500 次样本，胜率 60% → 几乎不收缩（因为数据充分）
```

SPAS 系统对所有信号的实际胜率都会做收缩估计，避免小样本带来的过度乐观。

## 1.3 期望值思维

概率正确不等于赚钱。你需要计算**期望值**：

```
期望值 = 赢的概率 × 赢时赚多少 - 输的概率 × 输时亏多少

例：某个策略胜率 55%，赢时赚 10%，输时亏 8%
期望值 = 0.55 × 10% - 0.45 × 8% = 5.5% - 3.6% = +1.9%

→ 正期望值！长期执行必然赚钱
```

> 💡 **胜率不是一切。** 一个胜率 30% 但赢一次赚 50%、亏一次只亏 5% 的策略，比胜率 70% 但每次只赚 2%、亏一次亏 15% 的策略好得多。

## 1.4 大数定律与小资金陷阱

大数定律说：只要期望值为正，交易次数越多越好。但这里有个致命陷阱——**你需要活到"大数"出现**。

```
策略胜率 55%，R:R = 1:1，期望值 10%

如果每次下注 50% 仓位：
  → 连续输 2 次仓位就只剩 25%
  → 可能等不到"大数"就爆仓了

如果每次下注 2% 仓位：
  → 连续输 10 次还剩 82%
  → 有足够时间让大数定律发挥作用
```

**仓位大小决定你能否活到大数定律生效。**

## 1.5 从概率到决策

三步法：
1. **算基率**：历史上类似情况成功概率是多少？
2. **贝叶斯更新**：有哪些新增信息要上调/下调概率？
3. **计算期望值并对比仓位**：交易的期望值是否足以让我分配仓位？

如果期望值为负或接近零 → 不交易，等待更好的机会。

## 本章要点

| 概念 | 说明 |
|------|------|
| 基率 | 不加入任何判断的历史成功率，无偏基准 |
| 贝叶斯更新 | 每次获得新信息后动态调整概率 |
| 收缩估计 | 小样本的极端值向均值收缩 |
| 期望值 | 胜率 × 赚多少 − 败率 × 亏多少 |
| 大数定律前提 | 仓位管理让你活到"大数"出现 |
