# 第二章：规则策略设计

## 2.1 规则策略 vs 主观判断

| | 规则策略 | 主观判断 |
|---|----------|----------|
| 一致性 | ✅ 永远按同一规则执行 | ❌ 情绪波动影响判断 |
| 可回测 | ✅ 可用历史数据验证 | ❌ 无法量化评估 |
| 可优化 | ✅ 参数可系统调优 | ❌ 只能靠经验积累 |
| 灵活性 | ❌ 对突发变化反应慢 | ✅ 可快速适应新情况 |

## 2.2 规则策略的基本结构

一条规则策略包含：

```
IF  条件A AND 条件B AND 条件C
THEN 执行买入
     止损设在 X%
     止盈设在 Y%
```

### SPAS 的规则引擎

SPAS 使用 **8 维规则向量** 来描述一个交易信号：

| 维度 | 示例值 | 含义 |
|------|--------|------|
| 市场状态 | BULL | 当前大盘方向 |
| 趋势强度 | 25 | ADX 值 |
| Setup 类型 | H2 | Wyckoff 形态 |
| Setup 质量 | 0.72 | 加权质量分 |
| 板块热度 | 0.45 | 消耗指数 |
| 资金流向 | +1 | 主力净流入 |
| K 线质量 | 0.68 | 6 维特征加权分 |
| 时间框架对齐 | +1 | 多层框架共振 |

## 2.3 规则设计的常见陷阱

### 陷阱一：过拟合（Overfitting）

```
错误做法：在 100 种规则组合中找最优的
  → 这组参数只是恰好迎合了历史数据
  → 未来大概率失效

正确做法：
  1. 用 70% 历史数据训练（训练集）
  2. 用 30% 历史数据验证（测试集）
  3. 训练集和测试集表现不能差太多
```

### 陷阱二：忽略交易成本

策略净收益 = 理论收益 - 佣金 - 印花税 - 滑点 - 冲击成本

一个高频策略如果每次只赚 0.1%，在 A 股扣除成本后大概率是负的。

### 陷阱三：幸存者偏差

历史回测时只测试了还活着的 ETF/股票，而退市/暴雷的没在数据里 → 高估了策略的真实收益。

## 2.4 规则策略的评估

| 指标 | 含义 | 及格线 |
|------|------|--------|
| 夏普比率 | 每单位风险获得的超额收益 | >0.8 |
| 最大回撤 | 历史上最大的连续亏损幅度 | <25% |
| 卡尔玛比率 | 年化收益 / 最大回撤 | >0.5 |
| 胜率 | 盈利交易占比 | >40% |
| 盈亏比 | 平均盈利 / 平均亏损 | >1.5 |

## 本章要点

| 概念 | 说明 |
|------|------|
| 规则策略 | IF-THEN 结构的交易逻辑 |
| SPAS 8 维向量 | 量化描述交易信号的 8 个维度 |
| 过拟合 | 参数恰好迎合历史数据 |
| 交易成本 | 不可忽略的五项成本 |
