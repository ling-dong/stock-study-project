# 第五章：市场情绪判断

## 5.1 市场情绪是什么？

市场情绪 = 市场参与者的整体心理状态。当大多数参与者都乐观时，市场往往到了顶部；当所有人都悲观时，底部可能已经不远。

> 💡 市场情绪是反向指标。极度乐观 = 即将回调，极度悲观 = 即将反弹。

## 5.2 情绪指标

### 恐惧贪婪指数

综合波动率、成交量、涨跌幅、期权等指标，输出 0-100：

| 值 | 含义 | 操作 |
|----|------|------|
| 0-25 | 极度恐惧 | 可以买入（别人恐惧时贪婪） |
| 25-45 | 偏恐惧 | 观望，等待更便宜 |
| 45-55 | 中性 | 正常交易 |
| 55-75 | 偏贪婪 | 减仓 |
| 75-100 | 极度贪婪 | 减仓或清仓（别人贪婪时恐惧） |

### 成交额异常
- 缩量下跌 → 市场冷清，买盘不足 → 可能继续跌
- 缩量上涨 → 跟风盘不够，上涨不健康
- 放量下跌 → 恐慌性抛售 → 近期大概率见底
- 放量上涨 → 增量资金入场 → 上涨健康

### 行业热度
某个行业连续大涨 + 成交量爆量 → 情绪过热 → 警惕回调。

## 5.3 极端情绪的反转信号

| 信号 | 说明 |
|------|------|
| 新开户数激增 | 散户大量涌入 = 顶部的标志 |
| 新基民疯狂追捧 | 新基金一日售罄 = 板块过热 |
| "这次不一样"叙事 | 最危险的四个字 |
| 媒体一致看多/空 | 舆论高度一致 = 随时反转 |

## 5.4 SPAS 的情绪采集

SPAS 的 `sentiment/collector.py` 从四个层次采集情绪数据：
1. 社交媒体（微博/雪球热词分析）
2. 新闻情感（标题和正文情绪倾向）
3. 交易数据（涨跌比、量比、换手率）
4. 资金流向（主力/散户净买入）

多源融合后才输出情绪评分，单源噪音大。
