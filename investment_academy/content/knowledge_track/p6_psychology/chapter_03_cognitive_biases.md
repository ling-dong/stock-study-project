# 第三章：认知偏差

## 3.1 什么是认知偏差？

认知偏差是人类大脑在处理信息时的系统性错误。**所有人都受其影响，你也不能例外。** 唯一的防御是知道它们的存在。

## 3.2 交易中四大认知偏差

### 确认偏差（Confirmation Bias）
```
只看支持自己判断的信息，忽略相反的证据。
"我已经看多半导体了 → 找 10 篇文章证明半导体好 → 忽略看空分析"
```
**应对**：在做交易决策前，刻意找一个反向观点来看。

### 锚定效应（Anchoring）
```
过度依赖第一个接触到的信息。
"我 5 块钱买的 → 现在 4 块 → '会回到 5 块' → 不止损"
买入价成了"锚"，让你无法接受亏损的现实。
```
**应对**：买入价不应该影响你的持仓决策。唯一该问的是"如果我现在是空仓，会不会在这个价位买入？"

### 损失厌恶（Loss Aversion）
```
亏损 1000 元的痛苦 ≈ 赚 2000-2500 元的快乐
人对损失的敏感度是对收益的 2-2.5 倍。
这会让你该止损时不止损，该止盈时过早止盈。
```
**应对**：用固定规则替代主观判断——止损止盈是数学，不是感觉。

### 近因偏差（Recency Bias）
```
最近发生的事对你的判断影响过大。
"最近三次 H2 都失败了 → H2 这个信号不行"
实际上历史胜率 55% 没变，只是最近不幸连亏。
```
**应对**：看大样本数据，不要被最近 3-5 次交易影响策略评估。

## 3.3 偏差检查清单

```
每次交易前问自己：
□ 我是不是只看了支持这个方向的信息？
□ 我的买入价在影响我的判断吗？
□ 我是不是因为最近赚/亏了所以改变了计划仓位？
□ 如果反过来做，有没有合理理由？
```

## 本章要点

| 偏差 | 表现 | 应对 |
|------|------|------|
| 确认偏差 | 只找支持自己的信息 | 刻意看反向观点 |
| 锚定效应 | 被买入价绑定 | 以空仓视角重新判断 |
| 损失厌恶 | 不止损/早止盈 | 数学替代感觉 |
| 近因偏差 | 被最近几笔影响判断 | 看大样本 |
