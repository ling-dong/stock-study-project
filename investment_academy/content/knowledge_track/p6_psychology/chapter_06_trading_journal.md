# 第六章：交易日志与复盘

## 6.1 为什么必须写交易日志？

> 不记录交易的人会不断犯同样的错，而且不知道自己犯过多少次。

职业交易员和业余交易者最大的区别之一：**前者有系统的交易记录和复盘习惯。**

## 6.2 每笔交易要记录什么？

| 字段 | 示例 |
|------|------|
| 日期 | 2026-07-15 |
| ETF 代码 | 512480.SH |
| Setup 类型 | H2 / L2 / FB / 趋势跟随 / 突破 |
| 入场理由 | "日线 EMA20 向上 + 60分 H2 形态 + 收盘位置 >0.6" |
| 出场理由 | "触达止盈位" / "止损触发" / "尾盘跳水主动离场" |
| 盈亏 % | +2.35% |
| 持仓天数 | 3 |
| 交易时情绪（1-5） | 3（正常） |
| 是否遵守计划 | ✅ / ❌ |
| 经验教训 | "入场时机晚了 30 分钟，错过了最优价格" |
| 是否错误交易 | ❌ |

## 6.3 从日志中发现自己的模式

积累 50-100 笔交易后，分析：

| 问题 | 发现 |
|------|------|
| 哪个 Setup 类型胜率最高？ | FB 胜率 40% → 少做或不做 |
| 什么时间段的交易最差？ | 下午 14:00-15:00 的交易胜率 35% → 下午不交易 |
| 什么情绪状态下亏损最多？ | 情绪分 4-5 分的交易胜率 30% → 情绪不好不交易 |
| 违反计划的频率？ | 每 10 笔有 3 笔违反 → 需要更严格的 Checklist |

## 6.4 错误模式库

将亏损交易分类，建立你的"常见错误"库：

```
错误模式 A：追涨被套
  → 买入时当日涨幅已 >3%
  → 防范：不买当日涨幅 >3% 的 ETF
  
错误模式 B：止损后反弹（"就差一点点"）
  → 止损设得太紧
  → 防范：ATR(14)×2 代替固定百分比止损
  
错误模式 C：盈利变亏损
  → 没有移动止盈
  → 防范：盈利 >5% 后设保本止损
```
