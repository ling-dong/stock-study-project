# 第二章：情绪识别与管理

## 2.1 交易中的五大情绪陷阱

### 1. FOMO（Fear Of Missing Out，害怕错过）

```
表现：看到别人赚钱、看到某 ETF 大涨 → 追高买入
结果：往往买在最高点，然后回调被套
```

**应对**：建立规则——永远不在当日涨幅 >5% 时追买。如果错过就错过，下一次机会一定有。

### 2. 恐慌（Panic）

```
表现：持仓突然下跌 → 恐惧蔓延 → 不计成本卖出
结果：往往卖在最低点，反弹时后悔
```

**应对**：提前设好止损位。到了止损位卖是计划内的事，不是情绪化操作。

### 3. 贪婪（Greed）

```
表现：已经盈利了 → 觉得还能涨 → 不舍得止盈
结果：利润回吐，大赚变不赚甚至亏损
```

**应对**：分批止盈。到第一目标位卖出 50%，剩余 50% 带移动止盈。

### 4. 复仇交易（Revenge Trading）

```
表现：刚才亏了一笔 → 不甘心 → 立刻加大仓位想"赚回来"
结果：连续亏损，单日亏损远超正常范围
```

**应对**：触发单日亏损 3% → 强制停止当日所有交易。这个规则是保命的。

### 5. Tilt 状态

Tilt 是一个扑克术语，指因情绪波动导致判断力丧失的状态。

```
Tilt 触发器：
  - 连续亏损 3 次以上
  - 止损后又反弹（"就差一点点！"）
  - 被市场"打脸"（判断完全相反）
  
Tilt 表现：
  - 违反交易计划
  - 加大仓位
  - 频繁交易
  - 忽视止损
```

> ⚠️ **识别 Tilt 状态是最重要的交易技能。** 大多数人亏损的最大原因不是策略不好，是在 Tilt 状态下做了不该做的交易。

## 2.2 情绪日志

每次交易后记录情绪状态（1-5 分）：

```
日期    操作      情绪(1-5)  触发因素
7/1     买入H2    4(焦虑)    连续3次止损后
7/3     卖出止盈  2(轻松)    按计划执行
7/5     追涨买入  5(冲动)    看到大涨怕错过(FOMO)
```

每周回顾：哪些情绪状态导致了好的决策？哪些导致了坏的决策？

## 2.3 情绪管理的实操方法

| 方法 | 操作 | 适用场景 |
|------|------|----------|
| **暂停 5 分钟** | 出现强烈交易冲动时，先离开屏幕 5 分钟 | FOMO / 复仇冲动 |
| **交易前 Checklist** | 每次交易前过一遍清单（见第 4 章） | 所有交易 |
| **仓位减半法** | 如果今天情绪不好，把计划仓位砍一半 | 焦虑/烦躁时 |
| **运动/睡眠** | 交易状态差时，先睡一觉或运动 30 分钟 | Tilt 状态 |

## 2.4 职业交易员的情绪曲线

```
开市 → 专注期（1-2h） → 疲劳期（午间） → 恢复期（下午） → 收市
         ↑                                        ↑
    情绪最稳定                                 情绪最容易波动
    重要决策在这时                              下午减少交易频率
```

> 🔑 每个交易员都有"情绪高峰期"和"情绪低谷期"。**在低谷期不交易，只观察。**
