# 第四章：纪律与交易规则

## 4.1 为什么纪律比天赋重要？

华尔街有一句老话：**"There are old traders and there are bold traders, but there are no old bold traders."**

翻译：有老交易员，有胆大的交易员，但没有老而且胆大的交易员。活得久的交易员靠的不是神预测，而是在该收手时收了手。

## 4.2 交易前 Checklist（每次交易必须逐项勾选）

### 市场环境检查

```
□ 大盘指数在 EMA(20) 之上还是之下？
  → 之上：正常仓位
  → 之下：半仓

□ 市场状态是 BULL / BEAR / NEUTRAL？
  → BULL：可以做多
  → BEAR：不做多
  → NEUTRAL：只做 1/4 仓以下的试探

□ ADX > 18 吗？
  → 是：趋势明确，可以交易
  → 否：震荡市，减少交易
```

### 入场条件检查

```
□ Setup 信号的质量评分 > 0.4 吗？
  → 如果是 H2/L2/FB，质量评分不够 → 不进场

□ 多时间框架方向一致吗？
  → 日线 + 60 分同时给出同向信号 → 可以
  → 只有一个框架有信号 → 不进场

□ R:R > 1.5 吗？
  → 计算止盈位和止损位 → 确认值不值
```

### 仓位检查

```
□ 仓位在 Kelly 0.25 范围内吗？
  → 单笔不超过 6-8% 总资金

□ 单 ETF 不超过 20% 吗？
  → 分散风险

□ 今天已经交易超过 3 笔了吗？
  → 超过则停止（高频 = 高情绪消耗 = 高错误率）
```

## 4.3 什么时候绝对不能交易

以下情况**禁止开仓**（不管你看到了多好的信号）：

| 情况 | 原因 |
|------|------|
| 刚止损后的 30 分钟内 | 情绪不稳定，容易复仇交易 |
| 连续亏损 3 次后 | 说明今天的判断有问题 |
| 身体不舒服/没睡好 | 认知能力下降 |
| 重大政策/数据发布前 30 分钟 | 新闻冲击不可预测 |
| 明显情绪波动的任何时刻 | 愤怒/兴奋/焦虑 = 判断力下降 |

## 4.4 周/月复盘模板

### 每周复盘

```
1. 本周盈亏：_________
2. 交易次数：_________
3. 盈亏比：_________
4. 最大回撤：_________
5. 是否严格遵守了交易前 Checklist？Y/N
6. 本周最大的情绪波动是什么？发生在哪笔交易？
7. 如果再给一次机会，本周哪笔交易我不会做？
```

### 每月复盘

```
1. 本月胜率 vs 历史胜率：_________
2. 本月盈亏比 vs 历史盈亏比：_________
3. 违反交易规则的次数：_________
4. Tilt 状态出现的次数和原因：_________
5. 下个月要做的一个改变：_________
```

## 4.5 建立"不准交易"清单

你自己的"绝对不能做的事"清单。示例：

```
❌ 止损后加仓
❌ 不设止损就进场
❌ 因为"感觉要涨"就买入
❌ 单日超过 3 次交易
❌ 下午开盘 30 分钟内重仓
❌ 在微信群/论坛看到别人赚钱后 FOMO 追买
```

> 💡 把这个清单贴在屏幕上。**每次交易前扫一眼。**
