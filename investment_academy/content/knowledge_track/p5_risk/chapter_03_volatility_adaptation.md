# 第三章：波动率适应

## 3.1 波动率的本质

波动率是**不确定性的价格**。高波动率意味着市场分歧大、方向性弱、随机性强。

> 波动率高的时候，同样方向判断正确的概率更低。**高波动率 = 降低仓位。**

## 3.2 波动率锚定

SPAS 使用波动率来动态调整仓位大小：

```
仓位缩放因子 = clamp(σ₀ / σ_now, 0.2, 2.0)

σ₀ = 历史平均波动率
clamp 限制缩放范围在 0.2~2.0 之间，避免极端值
```

## 3.3 ATR (Average True Range)

ATR 是最常用的波动率指标：

```
TR = max(High - Low, |High - Yesterday_Close|, |Low - Yesterday_Close|)
ATR(14)初始 = TR₁₄的SMA
ATR(14)后续 = (ATR(t-1)×13 + TR(t)) / 14
```
> 即 Wilder 平滑，等价于 EMA(α=1/14)

ATR 的绝对值告诉你价格一天正常波动多大。用它来设置止损：

```
止损位 = 入场价 - 2 × ATR(14)
```

> 💡 用 ATR 设止损比用固定百分比更科学。ETF 波动率变化时，止损宽度自动适应。

### 数值示例：科创 50 ETF 的 ATR 止损

假设科创 50 ETF（代码 588000）最近 14 日的 ATR(14) = 0.08 元，当前价格为 1.00 元：

- 2×ATR 止损宽度 = 2 × 0.08 = 0.16 元，相当于 **16% 的仓位风险宽度**。
- 若 ATR 萎缩到 0.03 元（市场平静），同样 2×ATR 止损宽度仅 6%，止损更紧凑。
- 若 ATR 放大到 0.15 元（高波动），止损宽度扩大到 30%，说明单笔风险过高，应先**缩小仓位**，而不是把止损拉得过宽。

> 实操要点：高波动期不是简单拉大止损，而是先降低仓位，使每笔亏损占账户的比例保持不变。

## 3.4 波动率周期

| 阶段 | 特征 | 操作 |
|------|------|------|
| 低波动率 | ATR 处于低位，市场沉寂 | 准备，即将突破 |
| 波动率上升 | 突破开始，ATR 攀升 | 可以参与，仓位谨慎 |
| 高波动率 | ATR 处于高位 | 减仓 |
| 波动率回落 | 行情明确，ATR 下降 | 加仓到正常水平 |

## 本章要点

| 概念 | 说明 |
|------|------|
| 波动率锚定 | σ_now vs σ₀ 决定仓位缩放 |
| ATR | 用真实波动范围设止损 |
| 波动率周期 | 低→升→高→降，仓位相应调整 |
