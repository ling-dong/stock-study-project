# 第一章：仓位管理

## 1.1 仓位管理比选股更重要

一个残酷的事实：**大多数交易者的亏损不是因为选错了股，而是因为下了太大的仓位。**

- 选错股但仓位小 → 可控亏损，下一次赚回来
- 选对股但仓位小 → 少赚，但不会亏
- 选错股且仓位大 → **致命打击，可能无法翻身**

> 💡 仓位管理是你在被证明错误之前，确保自己还能继续玩下去的唯一手段。

## 1.2 Kelly 公式

Kelly 公式是仓位管理的数学基础，它回答了"最优下注比例是多少"：

```
f* = (b × p - q) / b

其中：
f* = 最优下注比例
b  = 盈亏比（赢时赚多少 / 亏时亏多少）
p  = 胜率
q  = 1 - p
```

### 实例计算

```
策略胜率 p = 55%
盈亏比 b = 1.5（赢赚15%，亏亏10%）

f* = (1.5 × 0.55 - 0.45) / 1.5
   = (0.825 - 0.45) / 1.5
   = 0.375 / 1.5
   = 0.25 = 25%
```

Kelly 说最优仓位是 25% 总资金。**但实际执行中没人用满 Kelly，太激进了。**

## 1.3 Fractional Kelly — 实战版

SPAS 系统使用 **0.25 Kelly**（四分之一 Kelly）：

```
实战仓位 = 0.25 × f* = 0.25 × 25% = 6.25%
```

| Kelly 比例 | 特征 | 适用场景 |
|-----------|------|----------|
| 全额 Kelly | 最大理论收益 | 赌场，无限次下注 |
| 1/2 Kelly | 收益 75%，波动减半 | 高频交易 |
| 1/4 Kelly | 收益 50%，波动仅 25% | **A 股首选** |
| 1/8 Kelly | 极度保守 | 熊市或高波动期 |

> ⚠️ Kelly 公式的前提是"你知道真实的胜率和盈亏比"。在实战中你永远不可能精确知道，所以一定用 fractional，永远留足安全边际。

## 1.4 R:R 比（风险回报比）

```
R:R = 预期盈利 / 预期亏损

例：买入价 10 元，止盈 10.5 元（赚 5%），止损 9.8 元（亏 2%）
R:R = 5% / 2% = 2.5:1
```

| R:R | 最低胜率要求 | 实战可行性 |
|-----|-------------|-----------|
| 1:1 | 50% | 需要高频 + 低摩擦成本 |
| 1.5:1 | 40% | ETF 交易合理目标 |
| 2:1 | 34% | 趋势交易标准配置 |
| 3:1 | 25% | 可行但触发频率低 |

> 🔑 **永远不要在 R:R < 1 时入场。** 亏的时候亏得比赚的时候多，胜率再高长期也会亏钱。

## 1.5 金字塔加仓 vs 倒金字塔

### 金字塔加仓（✅ 推荐）

```
第一次：买入 50% 仓位 → 确认方向正确
第二次：买入 30% 仓位 → 趋势延续
第三次：买入 20% 仓位 → 趋势尾声不加太多
```

优点：均价低，回调时亏损可控。

### 倒金字塔（❌ 危险）

```
第一次：买入 20% → 涨了，后悔买少了
第二次：买入 40% → 还在涨，更后悔
第三次：买入 80% → 涨幅已经很大，回调来临 → 重仓亏损
```

## 1.6 固定比例仓位法（最简单也最有效）

每次交易固定使用总资金的 X%：
```
X = f(胜率, 盈亏比, Kelly 缩减系数)
  = (55% × 1.5 - 45%) / 1.5 × 0.25
  ≈ 6.25%
```

不管上次赚了还是亏了，每次都用 6.25%。这避免了"赢后膨胀/输后复仇"的心理陷阱。

## 本章要点

| 概念 | 说明 |
|------|------|
| Kelly 公式 | 最优下注比例的数学解 |
| Fractional Kelly | 实战用 1/4 Kelly，安全边际 |
| R:R 比 | 最少 >1.5 才考虑入场 |
| 金字塔加仓 | 先多后少 = 均价低 |
| 固定比例 | 最简单也最稳定的仓位法 |
