# 第五章：流动性管理

## 5.1 流动性的定义

流动性 = 能快速以合理价格买卖的能力。

| 指标 | 高流动性 | 低流动性 |
|------|----------|----------|
| 买卖价差 | <0.1% | >0.5% |
| 盘口厚度 | 买一卖一各有万手 | 买一卖一只有几手 |
| 日均成交额 | >5000 万 | <1000 万 |
| 冲击成本 | 几乎感觉不到 | 买入 100 手就能拉高 1% |

## 5.2 SPAS 的三层流动性检查

### 第一层：标的筛选（交易前）
- 只交易日均成交额 >5000 万的 ETF
- 排除流动性评分 <6 分的标的

### 第二层：执行检查（下单时）
- 单笔订单 < 前 5 分钟平均成交量的 10%
- 如果冲击成本估算 >0.2%，拆单执行

### 第三层：资金容量（组合层面）
- 单 ETF 资金不过组合的 20%
- 总仓位规模不超过单个 ETF 日均成交额的 5 倍

## 5.3 流动性陷阱

| 陷阱 | 表现 |
|------|------|
| 涨停买不到 | 排队买不进去 |
| 跌停卖不掉 | 排队卖不出来 |
| 尾盘流动性缺失 | 最后几分钟挂单价差极大 |
| 节假日前后 | 节前交易量萎缩，节后第一天冲击大 |

## 5.4 溢价/折价与流动性

ETF 流动性紧张时容易出现高溢价：
- **高溢价买入**：比实际价值多付钱 → 买方吃亏
- **高折价卖出**：比实际价值少拿钱 → 卖方吃亏
- 操作：溢价 >1.5% 不买，折价 <-1.5% 不卖

## 本章要点

| 概念 | 说明 |
|------|------|
| 流动性 | 快速合理价格买卖的能力 |
| 三层检查 | 标的筛选 → 执行 → 资金容量 |
| 冲击成本 | 大单对价格的影响 |
| 溢折价 | 偏离真实净值越多，流动性越差 |
