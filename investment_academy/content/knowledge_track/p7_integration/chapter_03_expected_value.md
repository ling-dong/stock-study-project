# 第三章：期望值计算

## 3.1 期望值 — 策略的体检报告

策略好不好，看四维指标：

```
❶ 胜率（Win Rate）
   → 盈利交易数 / 总交易数

❷ 盈亏比（Payoff Ratio）
   → 平均盈利金额 / 平均亏损金额

❸ 期望值（Expected Value）
   → 胜率 × 平均盈利 − (1−胜率) × 平均亏损

❹ 交易频率（Frequency）
   → 每月平均交易次数
```

### 四维组合

| 胜率 | 盈亏比 | 频率 | 评价 |
|------|--------|------|------|
| 高(>60%) | 高(>1.5) | 低 | 理想组合（但不常有） |
| 高 | 低 | 高 | 扣除成本可能亏钱 |
| 低 | 高 | 低 | 可以接受（趋势策略常见） |
| 低 | 低 | 任何 | 一定亏钱，立刻停止 |

## 3.2 含成本的期望值

裸期望值是乐观的。加入成本后：

```
真实期望值（每周期） = 裸期望值（每周期） - 每次交易成本 × 交易频率（每周期）

例：裸 EV = 2%/月，交易频率 = 10次/月，每次成本 0.1%
真实 EV = 2% - 10 × 0.1% = 1%/月
```

> 高频策略的成本侵蚀非常严重。每月交易 20 次、每次成本 0.1%，成本就会吃掉 2% 的收益。

## 3.3 SPAS 的成本模型

SPAS 的 `cost_model.py` 精确建模了 A 股交易成本：

```
佣金：0.025%（最低 5 元）
印花税：0.05%（仅卖出）
过户费：0.001%
滑点：1bp（正常）~ 10bp（流动性紧张时）
冲击成本：订单规模 / 日均成交额 × 5bp
```

## 3.4 评价你的策略

用**Brier 分数**衡量概率预测的质量：

```
Brier = Σ(预测概率 - 实际结果)² / n
Brier < 0.20：可用（金融领域基准较高）
Brier < 0.15：优秀
Brier < 0.10：非常优秀（金融领域极少达到）
```

用**夏普比率**衡量风险调整后收益：

```
夏普 > 1.5：非常好
夏普 > 1.0：好
夏普 > 0.8：合格
夏普 < 0.5：不合格
```

## 本章要点

| 概念 | 说明 |
|------|------|
| 四维指标 | 胜率 × 盈亏比 × 频率 × 成本 |
| 真实 EV | 裸 EV − 成本 × 频率 |
| Brier 分数 | 概率预测质量 |
| 夏普比率 | 每单位风险的收益 |
