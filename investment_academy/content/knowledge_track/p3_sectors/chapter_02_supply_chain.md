# 第二章：产业链分析

## 2.1 产业链的概念

产业链描述了从原材料到最终产品的完整价值传递过程。一个行业的景气度变化会沿着产业链向上游和下游传导。

## 2.2 产业链的传导效应

### 上游 → 中游 → 下游 传导

以半导体产业链为例：

```
上游（设备/材料）  →  中游（晶圆制造）  →  下游（芯片应用）
  刻蚀机/硅片            长鑫/中芯             手机/汽车/AI
      ↑                                              ↓
 最先受益于扩产                              最终需求决定一切
```

### 传导的时间差

| 环节 | 受益时序 | 弹性 |
|------|----------|------|
| 上游设备 | 最先（扩产开始即下单） | 中高 |
| 上游材料 | 其次（持续消耗品） | 中 |
| 中游制造 | 产能释放后才体现 | 中 |
| 下游应用 | 最后 | 低（成本端受损） |

## 2.3 SPAS 产业链图谱

SPAS 系统维护了包含 60+ 条产业链边的行业关联图。关键关联示例：

```
石油 → 化工 → 化纤 → 纺织服装
锂矿 → 正极材料 → 电池 → 新能源车
芯片设计 → 晶圆制造 → 封测 → 消费电子
```

> 🔑 当一个上游行业大涨时，关注它的中下游关联行业。传导可能滞后 1-3 个月。

## 2.4 产业链分析的实战价值

1. **提前布局**：上游订单暴增 → 3 个月后中游业绩兑现
2. **风险预警**：下游需求萎缩 → 向上传导将影响整条链
3. **套利机会**：上下游估值裂口过大 → 均值回归
